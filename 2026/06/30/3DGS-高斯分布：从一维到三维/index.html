[3DGS]高斯分布：从一维到三维

日期：2026-06-26
内容：概率基础 + 一维高斯 + 三维高斯几何理解

一、概率基础速览

1.1 PDF vs CDF

  • PDF（概率密度函数）：描述随机变量在某值附近的"可能性密度"。不是概率本身，曲线下总面积 = 1
  • CDF（累积分布函数）：x 处 CDF 值 = PDF 从 -∞ 到 x 的积分，单调递增 0→1
分布 PDF 特征 3DGS 关联
正态 N(0,1) 钟形，中心对称 核心：3DGS 的高斯基元就是 3D 高斯
均匀 U(-2,2) 常数（区间内） 从不使用——高斯比均匀更"平滑可微"
指数 Exp(1) 单调递减 不透明度衰减的参考模型

1.2 期望与方差的几何意义

1.3 为什么 3DGS 用高斯

  1. 可微性：高斯函数处处可微 → 能做梯度下降
  2. 紧凑支撑：99.7% 的质量在 μ±3σ 内 → 便于裁剪（核心性能优化）
  3. 闭式积分：高斯在任意区间上的积分有解析式
  4. 仿射变换保持高斯性：高斯的线性变换仍是高斯

二、一维高斯分布（demo_02）

2.1 数学形式

$$\displaystyle p(x) = \frac{1}{\sigma\sqrt{2\pi}} \exp\left(-\frac{(x-\mu)2}{2\sigma2}\right)$$

2.2 公式拆解（不用概率论理解）

把公式拆成两块看：

B 的作用：产生钟形

  • (x-μ)² 保证了左右对称
  • ÷σ² 保证了 σ 越大衰减越慢（曲线越宽）

A 的作用：归一化

σ 越大 → 曲线越宽 → 如果高度不变，总面积会膨胀 → A 自动压低高度，总面积始终 = 1

2.3 μ 和 σ 的几何含义

μ（均值）= 钟形曲线的中心位置

  • 改 μ = 把整条曲线水平平移，形状完全不变

σ（标准差）= 钟形曲线的胖瘦

  • σ 小（如 0.5）→ 瘦高（数据集中）
  • σ 大（如 2.0）→ 矮胖（数据分散）

2.4 68-95-99.7 规则

区间 概率 3DGS 中的意义
μ ± 1σ 68.3% 高斯基元的"核心区"
μ ± 2σ 95.5% 绝大部分贡献
μ ± 3σ 99.7% 裁剪阈值——超出 3σ 忽略不计

2.5 高斯函数的可微性

对 μ 的梯度：
$$\displaystyle \frac{\partial p}{\partial \mu} = p(x) \cdot \frac{x-\mu}{\sigma^2}$$

  • x > μ 时：梯度为正（增大 μ 靠近 x → 概率增大）
  • x < μ 时：梯度为负
  • x = μ 时：梯度为 0

对 σ 的梯度：
$$\displaystyle \frac{\partial p}{\partial \sigma} = p(x) \cdot \left(\frac{(x-\mu)2}{\sigma3} - \frac{1}{\sigma}\right)$$

  • |x-μ| = σ 处：梯度 = 0（拐点）
  • |x-μ| < σ 时：缩小 σ 让分布更高
  • |x-μ| > σ 时：增大 σ 让分布更宽

三、三维高斯（3DGS 的核心）

3.1 它长什么样

3.2 四个参数控制一个高斯

参数 符号 几何含义 类比
位置 μ (xyz) 椭球体中心在 3D 空间中的坐标 物体的位置

不透明度 α 中心处"浓度"，1=全不透明，0=全透明 物体的浓淡

3.7 3σ 裁剪——性能关键

到中心距离 G(X) 值 3DGS 处理
0σ 1.000 中心，最亮
1σ 0.607 约 60%
2σ 0.135 约 14%
3σ 0.011 约 1%，临界点
>3σ <0.01 直接跳过不计算

五、核心结论

  1. 一维高斯 = 钟形曲线：μ 定位置，σ 定宽窄
  2. 3DGS 的高斯 = 空间中的模糊椭球体：μ 定位置，Σ 定形状/方向/大小
  3. 3σ 规则是关键优化：超过 3σ 不计算，才能实时渲染
  4. 高斯处处可微：这使梯度下降能直接优化所有参数
  5. α-blending 混合：从近到远叠加所有高斯，算出最终像素颜色
