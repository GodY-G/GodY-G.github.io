1.3 三个核心机制

机制 作用 关键参数
动量 m 平滑梯度方向，消除震荡 β₁=0.9，历史占90%
自适应 v 每参数独立调步长，陡坡小步平路大步 β₂=0.999，有效lr=α/√v̂
偏差修正 修正初始 m=v=0 导致的早期低估 t=1 时修正因子=1/(1-β₁)=10

二、损失函数

2.1 L1（MAE）— 绝对值

特点：对大小误差一视同仁，不敏感异常值（如镜面反射）。

2.2 L2（MSE）— 平方

特点：对大误差惩罚极重（平方放大），对小误差几乎无视。一个异常像素就能绑架整个优化。

误差 L1 贡献 L2 贡献
0.5 0.5 0.25
0.1 0.1 0.01
0.02 0.02 0.0004

2.3 SSIM — 结构相似度（三因子）

因子 计算 衡量什么
亮度 l (2μₓμᵧ+C₁)/(μₓ²+μᵧ²+C₁) 整体亮度是否一致
对比度 c (2σₓσᵧ+C₂)/(σₓ²+σᵧ²+C₂) 纹理强弱是否一致
结构 s (σₓᵧ+C₃)/(σₓσᵧ+C₃) 纹理模式是否一致

SSIM 的核心洞察：不看单个像素，看局部窗口（11×11）的统计量。

例子——整体亮度偏移：

  • 真图：平均像素 0.5。渲染：平均像素 0.6（偏亮 0.1）
  • L2：每个像素 0.1² = 0.01 → 大惩罚
  • SSIM：l ≈ (2×0.5×0.6)/(0.25+0.36) ≈ 0.98 → 几乎不惩罚

因为人眼对整体亮度偏移不敏感，SSIM 也不敏感——这恰好是符合感知的评估方式。

2.4 3DGS 组合损失

  • 只用 L1 → 模糊（颜色对但纹理没了）
  • 只用 SSIM → 结构对但可能偏色
  • 组合：颜色准 + 纹理清晰 + 边缘锐利
