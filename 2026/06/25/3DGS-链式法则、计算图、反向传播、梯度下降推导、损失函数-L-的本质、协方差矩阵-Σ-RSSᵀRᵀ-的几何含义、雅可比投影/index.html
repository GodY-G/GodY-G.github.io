病态条件 → Adam 的动机

不同参数方向的梯度量级可能差 100 倍（如 f=0.5(x²+100y²)中 ∂f/∂x=x 量级~1，∂f/∂y=100y 量级~100）。单一学习率无法兼顾 → Adam 自适应调整每个参数的有效学习率。

三、损失函数 L — 核心理解

L 是什么？

L 是百万维函数：输入是 N 个高斯的全部参数（μ,q,s,α,sh），输出是一个标量——渲染图和真实照片的差异。

为什么 L 越小渲染越好？

L₁↓ = 颜色更准 | (1-SSIM)↓ = 结构更对 → 渲染图更接近真图。

四、协方差矩阵与投影

Σ=RSSᵀRᵀ 为什么是协方差矩阵？

  • S² = diag(s₁²,s₂²,s₃²) 是正放的椭球（半轴长 s₁,s₂,s₃，主轴沿坐标轴）
  • R 旋转该椭球 → 得到歪的椭球
  • 数学：Σ = R·S²·Rᵀ = 先通过 Rᵀ 把向量转到主轴方向 → 在主轴方向拉伸 S² → 再用 R 转回来
  • Σ 天然对称且正定 → 合法协方差矩阵

协方差矩阵的几何含义

等密度面 (x-μ)ᵀΣ⁻¹(x-μ)=1 是椭球面。特征向量=椭球主轴方向，特征值=半轴长²。
