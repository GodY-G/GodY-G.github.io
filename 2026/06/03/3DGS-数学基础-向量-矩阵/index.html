向量——点积与叉积

1 先理清：什么是向量？

你已经会在 Python 中写 np.array([x, y, z])。但这只是向量的数据表示。向量本身到底是什么？

向量的两种视角：

视角 理解 例子
几何 空间中一个有方向、有长度的箭头 从原点指向点 (2, 3, 5)
代数 一组有序数 (x, y, z) np.array([2, 3, 5])

点积告诉你什么？

判断规则：

点积 cos(θ) θ 范围 含义
> 0 > 0 0° ~ 90° 大致同向
= 0 = 0 恰好 90° 垂直
< 0 < 0 90° ~ 180° 大致反向

最重要的几何直觉：投影

a 在 b 方向上的投影长度 = (a·b) / |b|

画在纸上：a=(3,2) 是一个从原点出发指到 (3,2) 的箭头。把它投影到 x 轴（即 b 的方向），投影点就是 (3,0)。投影长度 = 3 恰等于 a 在 x 轴上的分量。

这就是投影的全部：a·b̂ 算出”a 在 b 指向的方向上走了多远”。

3DGS 中怎么用点积？

场景1：判断高斯在不在相机视野里

相机的视线方向是 d（单位向量），从相机到高斯的向量是 v。如果 d·v > 0，说明高斯在相机前方；如果 < 0，在身后（扔掉不渲染）。

场景2：球谐函数计算

球谐函数本质是视线方向 d 和球谐基函数 Y_lm 的某种”方向相似度”计算，内积形式。

3DGS 中怎么用叉积？

计算平面法向量：建筑场景中，给定墙面上的三个点 p1, p2, p3，墙面的法向量方向 = (p2-p1) × (p3-p1)。这对于判断高斯椭球是否应该”躺平”贴在墙面上非常关键。

3 六种基本变换——对照 demo_01 理解

demo_01 的 demo_matrix_as_transform() 函数展示了六种矩阵对单位正方形的影响。单位正方形 [0,0]→[1,0]→[1,1]→[0,1] 是观察变换的标准参考物。

变换1：单位矩阵——不变

正方形纹丝不动。就像乘以 1。

变换2：水平拉伸 ×2

效果：正方形在 x 方向被拉成原来的 2 倍宽度。det(A) = 2×1 = 2，面积变为原来的 2 倍。

变换3：垂直拉伸 ×2

效果：正方形在 y 方向变高。det = 2。

变换4：旋转 90°（逆时针）

效果：整个正方形绕原点转了 90°。det = 0×0 - (-1)×1 = 1，旋转不改变面积。

变换5：水平剪切

效果：正方形变成平行四边形——顶边向右偏了 0.5。注意：det = 1×1 - 0.5×0 = 1，剪切不改变面积！

变换6：复合变换

det = 2×1.5 - 0.5×0.3 = 3 - 0.15 = 2.85，面积扩大了 2.85 倍。
